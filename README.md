## üõ† How to Run Locally

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/furniqo.git
```

### 2. Install Dependencies
```bash
# Install frontend
cd furniqo
npm install

# Install backend
cd ../backend
npm install

# Install 3D (optional)
cd ../furniqo-3D
npm install (only if needed)
```

### 3. Run the Project

Use a single command via `concurrently`:

```bash
npm run dev:all
```

Or manually run:
```bash
# Frontend
cd furniqo
npm run dev

# Backend
cd ../backend
npm run dev

# 3D Server
cd ../furniqo-3D
cd furniqo-3d
python -m http.server
```
